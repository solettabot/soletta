43 /* flow-builder is a helper component that can be used to
44  * create nodes and connections specs with a friendly API.
45  * So instead of declaring a node spec structure and an ordered
46  * connection spec structure directly, using node and port indexes,
47  * builder can be used to add nodes and connections in a programmatic way,
48  * using nodes and ports names.
49  * This approach should led to a code easier to read and maintain, but
50  * it would consume a bit more memory and processing.
51  */
